# 7. トロフィー

## コンセプト
東京で行った賢い選択を、競争ではなく自分自身の実績として残す案です。時間、快適さ、家族、地域貢献、公共施設など多様な行動を評価します。

## 検証したいこと
- トロフィーがROIの順位付けではなく、行動の多様性を記録するものとして伝わるか
- 解除演出が行動後の振り返りを促すか
- 未解除の実績が探索のきっかけになるか

## 主なフロー
1. トロフィー解除の可能性がある候補を選ぶ
2. 行動後の成果を確認する
3. 新しいトロフィーの解除・進捗を確認する
4. コレクション画面でカテゴリ別の実績を見る

## 表示する価値
ブロンズ、シルバー、ゴールド、プラチナのランクを、色だけでなく形とラベルでも区別します。隠し実績はヒントのみを表示します。

## モック範囲
トロフィー進捗と最後に解除した実績はローカルストレージへ保存します。実際の行動データ連携はありません。
