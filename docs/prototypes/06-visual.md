# 6. アイコン中心

## コンセプト
設定以外の文章を極力減らし、アイコン、数値、色、ゲージ、地図風の表現で選択を支援する案です。

## 検証したいこと
- 文章を読まずに目的や候補の違いを理解できるか
- 数値とピクトグラムだけで、時間・費用・混雑・満足度を比較できるか
- 都市貢献を視覚的に把握できるか

## 主なフロー
1. アイコンで目的や気分を選ぶ
2. 視覚的な分析演出を見る
3. カルーセル形式で候補を比較する
4. 選択後に時間・費用・都市ポイントの変化を見る

## 表示する価値
混雑は人型アイコン、満足度は表情、都市貢献はリングや街のアイコンで表現します。

## モック範囲
候補・ルート・スコア・履歴は固定データです。実際の地図・交通情報・混雑情報とは連携しません。
