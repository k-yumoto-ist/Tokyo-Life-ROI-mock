# 5. 都市貢献

## コンセプト
自分にとって合理的な選択が、混雑分散や公共施設の活用にもつながることを、軽いゲーム要素で可視化する案です。

## 検証したいこと
- 都市への副次的な価値を押し付けずに伝えられるか
- 候補選択後にだけ貢献スコアを公開する体験が機能するか
- バッジやポイントが継続利用の補助になるか

## 主なフロー
1. 今日のクエストを選ぶ
2. 公共施設・公園・地域施設などの候補を確認する
3. 自分で一案を選ぶ
4. 個人ROIと都市貢献、東京ポイント、バッジを確認する

## 表示する価値
選択前は混雑の少なさ、移動負担、公共施設の活用などの定性的なヒントのみを表示します。

## モック範囲
都市貢献レベル、ポイント、バッジ、候補施設は固定データです。行政制度や実在ポイント制度との連携はありません。
