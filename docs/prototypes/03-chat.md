# 3. AI相談型

## コンセプト
自然文で今日の希望を伝え、AIが条件をチップとして整理してから候補を提案する案です。入力フォームより会話に近い導入を検証します。

## 検証したいこと
- ユーザーが条件を言語化しやすいか
- AIが読み取った条件が見えることで安心できるか
- 条件の見落としをチップで修正できるか

## 主なフロー
1. 自然文で希望を入力する
2. 家族、時間、混雑回避などの条件チップを確認する
3. おすすめ候補を確認する
4. 候補を選び、行動・振り返りへ進む

## 表示する価値
会話文そのものではなく、整理された条件と短い提案理由を中心に表示します。

## モック範囲
AI応答は固定データです。生成AI APIや外部検索には接続しません。
